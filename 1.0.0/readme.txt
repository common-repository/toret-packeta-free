== Description ==
WooCommerce intergration plugin to connect to Packeta services.
Plugin allows send WooCommerce order data to Packeta system. That means that you do not neet to manually copy order details for creating a package but plugin automate it.
For connecting to Packeta services you need to have a Packeta account.
